# tasklet-apps

Monorepo for [Tasklet](https://tasklet.ai) instant apps — auto-synced dashboards and tools.

## Apps

| App | Description |
|-----|-------------|
| [pr-dashboard](apps/pr-dashboard/) | Jules PR automation dashboard — repo health, merge stats, activity feed |

## Structure

```
apps/
└── pr-dashboard/     # Jules PR Dashboard
```

Each app lives under `apps/` and follows the Tasklet instant app conventions (React + DaisyUI + TypeScript).
